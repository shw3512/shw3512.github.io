Final exam topics

Caveat lector : I have tried to be thorough, but this list may not be comprehensive. If you don't see a topic you expect, please ask me in class on Wednesday.

  • 1st-order ODEs
    • How to solve
    • How to analyze : graphical analysis, existence and uniqueness (Picard's theorem), key ideas behind numerical approximation
  • Equilibrium solutions : how to find, how to analyze stability
  • Linearity (a.k.a. "nonhomogeneous principle")
    • 1st-order ODEs : integrating factor, variation of parameters
    • 1st-order systems : undetermined coefficients, variation of parameters
  • Linear algebra, especially...
    • Solving systems of linear equations
    • Rank-nullity theorem
    • Eigenvalues, eigenvectors, eigenspaces : definitions and how to compute
    • How to diagonalize a matrix (when it is possible)
  • Matrix exponential : properties, general idea of how to compute it (particularly, how it relates to eigenvalues and eigenvectors of the original matrix)
  • Solving 1st-order systems
    • How to find the general solution
    • How to translate higher-order linear ODE ↔ 1st-order linear system of ODEs
    • Phase plane
    • See also "Linearity" above
  • How to solve higher-order ODEs
    • Translate into a 1st-order linear system
    • Jump straight to the characteristic polynomial
    • Use the laplace transform
  • Linearization
    • How to find equilibria (e.g., using nullclines)
    • How to analyze stability of equilibria (using the jacobian matrix)
    • How to (roughly (!)) sketch phase plane
  • Laplace transform : definition, how to use it to solve higher-order ODEs
